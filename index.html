Nicolas Le Jeune.

Je développe des sites web,
et des jeux-vidéo pour le fun !

Je suis un développeur web junior passionné par la création d'expériences interactives en ligne. Avec une solide formation en développement d'applications JavaScript React et une expérience dans la création de jeux vidéo en tant qu'amateur, je suis constamment à la recherche de nouvelles opportunités pour développer mes compétences et mettre en pratique mes connaissances.

Ma formation en développement d'applications JavaScript React avec OpenClassrooms m'a permis d'acquérir une solide base de connaissances et de compétences dans le domaine du développement web. Je suis à l'aise avec les langages de programmation HTML, CSS et JavaScript, ainsi qu'avec l'utilisation du framework React.
